## Release History

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Enhanced agent memory system with persistent context across sessions
- New collaborative mode allowing multiple agents to work together on complex tasks
- Improved API integration with automatic retry logic and better error handling
- Added support for custom model fine-tuning directly in the platform

## Improvements

- Faster response times for conversational agents through optimized prompt processing
- Better playground performance with improved loading times and real-time updates
- Enhanced security with improved API key management and encryption
- Updated user interface with cleaner design and improved accessibility

</Update>

<Update label="2024-09-20" description="v2.0.0" tags={["breaking", "feature"]}>

## Breaking Changes

- Updated agent configuration format to support advanced features
- Changed webhook payload structure for better consistency
- Modified API authentication to use JWT tokens instead of API keys

## New Features

- Complete redesign of the agent builder with drag-and-drop interface
- Introduction of generative tools library for common AI tasks
- Multi-agent orchestration system for complex workflows
- Advanced analytics dashboard for monitoring agent performance

</Update>

<Update label="2024-08-10" description="v1.5.0" tags={["feature", "bugfix"]}>

## New Features

- Added support for webhook integrations
- New agent templates for common use cases like customer support and content generation
- Improved playground with better debugging tools
- Added export/import functionality for agent configurations

## Bug Fixes

- Fixed issue with agent responses timing out on long conversations
- Resolved memory leaks in long-running agent sessions
- Corrected API rate limiting calculations
- Fixed webhook signature verification problems

</Update>
